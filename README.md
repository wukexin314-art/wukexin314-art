## 🌟 Featured Project

### 🦟 [Nowcast Dengue in India](https://github.com/wukexin314-art/Nowcast-Dengue-in-India)

A forecasting and nowcasting project designed to estimate recent dengue trends under reporting delays by combining **WHO data, Google Trends, Wikipedia signals, and time-series modeling**.

- 🌷 Built a two-stage framework with proxy learning and autoregressive nowcasting  
- 📌 Automated data updating and preprocessing workflows  
- 📊 Developed dashboard and API outputs for communicating current estimates  
- 💡 Focused on practical forecasting under incomplete and lagged observations
